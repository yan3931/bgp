V3.1 静态权重计算 (Softplus 平滑)：

$$W^* = \left[ 0.2 \times \frac{R-5}{5} + 0.8 \times \left(\frac{C}{5}\right)^2 \right] \times \log_2\left(1 + \frac{T}{15}\right)$$ $$W_{static} = \tau \ln(1 + e^{W^*/\tau}), \quad \tau = 0.1$$

PvP 综合胜率（对数优势模型）：

$$\hat{p}_g = \frac{w_g + \lambda b_g}{n_g + \lambda}, \quad A_g = \text{logit}(\hat{p}_g) - \text{logit}(b_g)$$ $$\omega_g = W_{static} \times \frac{n_g}{n_g + k}, \quad P_{final} = \sigma\left(\text{logit}(0.25) + \frac{\sum \omega_g A_g}{\sum \omega_g}\right)$$

PvE 综合通关率：

$$P^{(pve)}_{final} = \frac{\sum \omega_g \hat{p}_g}{\sum \omega_g}, \quad k_{pve}=10$$
